Contributors: Jackson Laycock
Requires at least: 2.6
Tested up to: 2.9.1

Twitpic Expander scans posts for twitpic URL's and automatically expands them and shows an image.  

== Details ==

Transforms plain text url's into html image tags.


== Installation ==

1. Download the plugin archive and expand it (you've likely already done this).
2. Put the twitpic-expander.php into your wp-content/plugins/ directory.
3. Go to the Plugins page in your WordPress Administration area and click 'Activate' for Twitpic Expander.


== Configuration ==

There is no configuration options, but to display thumbnails simply change the code http://twitpic.com/show/large/{$match} to http://twitpic.com/show/thumb/{$match}
